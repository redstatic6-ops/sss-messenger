# 🚀 Финальные шаги для сборки EXE

## ✅ Что уже сделано:

1. ✅ Веб-приложение собрано (папка `dist/`)
2. ✅ Electron настроен с кастомным окном (без рамки браузера)
3. ✅ Создан установщик NSIS с уникальным дизайном
4. ✅ Добавлена кастомная панель управления окном
5. ✅ Настроен системный трей
6. ✅ Блокировка множественных экземпляров
7. ✅ Автоматические разрешения для микрофона/камеры

## 📝 Что нужно сделать СЕЙЧАС:

### Шаг 1: Создать иконку

**Вариант A (Быстрый):**
1. Открыт файл `build/icon-generator.html` в браузере
2. Нажмите кнопку "Скачать PNG 256x256"
3. Перейдите на https://convertio.co/ru/png-ico/
4. Загрузите скачанный PNG
5. Конвертируйте в ICO (выберите все размеры: 16, 32, 48, 64, 128, 256)
6. Скачайте ICO файл
7. Сохраните как `build/icon.ico`

**Вариант B (Если есть своя иконка):**
- Просто положите свой файл `icon.ico` в папку `build/`

### Шаг 2: (Опционально) Удалить строки с sidebar из package.json

Если не хотите создавать изображения для боковой панели установщика, удалите эти строки из `package.json`:
```json
"installerSidebar": "build/installer-sidebar.bmp",
"uninstallerSidebar": "build/installer-sidebar.bmp",
```

### Шаг 3: Собрать EXE

Выполните команду:
```bash
npm run electron:build
```

### Шаг 4: Найти готовый установщик

После успешной сборки в папке `dist-electron/` найдёте:
- `SSS Messenger-Setup-1.0.0.exe` - ГОТОВЫЙ УСТАНОВЩИК! 🎉

## ⚠️ Возможные ошибки:

**"icon.ico not found"**
- Создайте icon.ico в папке build/ (см. Шаг 1)

**"installerSidebar.bmp not found"**
- Удалите строки с sidebar из package.json (см. Шаг 2)

**Ошибка при сборке**
- Убедитесь, что `npm run build` выполнен успешно
- Проверьте, что папка `dist/` существует
