### 🎨 Для создания кастомной иконки:

Тебе нужно создать файл `build/icon.ico` (256x256 пикселей).

Можешь использовать онлайн конвертер:
- https://convertio.co/png-ico/
- https://icoconvert.com/

Или создай сам в любом графическом редакторе.

### 📦 Параметры установщика (уже настроено):
- Выбор директории установки
- Ярлык на рабочем столе
- Ярлык в меню Пуск
- Кастомная иконка (когда добавишь icon.ico)
- Лицензия MIT

## Сборка установщика:

```bash
npm run electron:build
```

Установщик будет в папке `dist-electron/SSS Messenger Setup 1.0.0.exe`

## Что можно добавить (опционально):

1. **build/icon.ico** - иконка приложения 256x256
2. **build/installerHeader.bmp** - баннер сверху установщика (150x57)
3. **build/installerSidebar.bmp** - боковой баннер (164x314)

Если не добавишь изображения, установщик будет работать с дефолтными.
