# SSS Messenger - Сборка приложения

## Подготовка к сборке

### 1. Создание иконки (icon.ico)

Вам нужно создать файл `build/icon.ico` с разрешениями: 16x16, 32x32, 48x48, 64x64, 128x128, 256x256.

Можете использовать онлайн конвертер:
- https://convertio.co/ru/svg-ico/
- https://icoconvert.com/

Загрузите `build/icon.svg` и конвертируйте в ICO формат со всеми размерами.

### 2. (Опционально) Создание изображений для установщика

Создайте изображения для боковой панели установщика:
- `build/installer-sidebar.bmp` - 164x314 пикселей
- Можете использовать любой графический редактор

Если не хотите создавать, удалите эти строки из package.json:
```
"installerSidebar": "build/installer-sidebar.bmp",
"uninstallerSidebar": "build/installer-sidebar.bmp",
```

## Сборка приложения

### Шаг 1: Установка зависимостей (если еще не установлены)

```bash
npm install
```

### Шаг 2: Сборка веб-приложения

```bash
npm run build
```

### Шаг 3: Сборка Electron приложения и создание установщика

```bash
npm run electron:build
```

## Результат

После успешной сборки в папке `dist-electron` появятся:

- `SSS Messenger-Setup-1.0.0.exe` - установщик приложения
- Другие файлы для распространения

## Установка приложения

1. Запустите `SSS Messenger-Setup-1.0.0.exe`
2. Следуйте инструкциям установщика
3. Выберите папку установки (по умолчанию: C:\Program Files\SSS Messenger)
4. Установщик создаст ярлыки на рабочем столе и в меню Пуск

## Особенности приложения

- ✅ Нет рамки браузера - кастомная панель управления окном
- ✅ Единственный экземпляр - нельзя запустить дважды
- ✅ Системный трей - сворачивание в трей
- ✅ Автоматические разрешения для микрофона/камеры
- ✅ Поддержка протокола sss:// для открытия ссылок
- ✅ Блокировка DevTools в релизе
- ✅ Профессиональный установщик NSIS

## Решение проблем

### Ошибка "icon.ico not found"
Создайте файл icon.ico в папке build/

### Ошибка при сборке
Убедитесь, что:
- Node.js версии 16 или выше
- Все зависимости установлены
- Папка dist/ существует после `npm run build`

### Приложение не запускается
- Проверьте, что Supabase настроен правильно
- Убедитесь, что файлы в dist/ не повреждены
