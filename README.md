⬇️ Скачать и установить
Ничего собирать не нужно — готовые сборки лежат в разделе Releases.

Платформа	Что скачать	Ссылка
🪟 Windows	SSS-Messenger-Setup-*.exe	→ Последний релиз
🤖 Android	app-debug.apk	→ Скачать APK
🌐 Веб	—	открывается прямо в браузере
Windows: запустите установщик и следуйте мастеру. Если SmartScreen покажет предупреждение — «Подробнее» → «Выполнить в любом случае» (сборка без подписи).

Android: скачайте APK, при установке разрешите «Установка из неизвестных источников» для браузера/проводника.

🔄 Авто-обновление. Приложение само проверяет новые версии: десктоп обновляется через electron-updater, Android предлагает скачать свежий APK. Обновления выходят прямо в Releases.
